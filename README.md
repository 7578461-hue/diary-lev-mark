# Lev & Mark — Twins Tracker

Мобильное приложение для контроля ухода за близнецами Лев и Марк
(09.01.2026). Двуязычное (RU/EN), для нянь-филиппинок и родителей.

Стек: одностраничное PWA (`index.html`) + Supabase (PostgreSQL +
Storage + Realtime). Без сборки, без зависимостей.

---

## Что умеет

**4 вкладки, нижняя навигация:**
- 🍼 **Кормление / Feeding** — смесь KABRITA + NUTRILON, вода, стул
  (цвет + консистенция), фото, голосовые и текстовые заметки,
  дневная сводка, итог дня от няни
- 💊 **Лекарства / Meds** — 3 карточки (Утро/День/Вечер) с
  Мальтофером и Фолиевой кислотой, авто-фиксация времени приёма
- 🥦 **Прикорм / Solids** — продукт, объём, реакция, фото, заметки
- 📊 **История / History** — лента по дням + профиль "К врачу"
  с графиками питания/стула/веса и процентом выполнения лекарств

**Два режима в шапке:**
- 👶 Няня / Nanny — формы ввода
- 👨‍👩‍👦 Родители / Parents — только чтение

Шрифт: Inter. Цвета: Лев — синий `#2A6B88`, Марк — коричневый `#7A5030`.

## Деплой за 3 шага

### Шаг 1 — Supabase (база данных + хранилище фото)

1. Если ещё нет проекта — открыть **supabase.com** → New project,
   регион **Europe (Frankfurt)**. Если проект уже есть (URL
   `oqlnsqtveilnphvchwtn.supabase.co` уже вшит в `index.html`) —
   пропустить.
2. SQL Editor → New query → вставить содержимое `supabase-setup.sql`
   → **Run**. Скрипт идемпотентный, можно запускать повторно.
3. Storage → **New bucket** → имя `baby-photos` → **Public: ON** → Create.
   Без публичного bucket'а фото не будут грузиться.

> **Если хочешь свой Supabase-проект**, замени в `index.html`
> константы `SUPABASE_URL` и `SUPABASE_KEY` (строки ~640).

### Шаг 2 — Деплой на Netlify

Самый простой способ — **Netlify Drop** (даже регистрироваться не нужно
для одноразового деплоя, но лучше создать аккаунт чтобы получить
постоянную ссылку):

1. Открыть **https://app.netlify.com/drop**
2. Перетащить **папку `diary-lev-mark`** (целиком) в окно
3. Netlify выдаст ссылку вида `https://random-name-123.netlify.app`

Альтернатива (Vercel): открыть **vercel.com** → Add New → Project →
Deploy from your computer → выбрать папку.

### Шаг 3 — Раздать ссылку

- Ссылка работает на iPhone и Android без установки
- На iPhone: открыть в Safari → "Поделиться" → "На экран Домой" —
  получится иконка как у нативного приложения
- Никаких аккаунтов и паролей: данные пишутся в общую базу

## Кастомный домен

В Netlify → Site settings → Domain management → Add custom domain.
Например: `lev-mark.ru` или `babies.ferdmans.ru`.

---

## Обновление приложения

Изменил `index.html`? Снова перетащи папку на Netlify Drop —
получишь новую ссылку. Чтобы обновить ту же ссылку: в Netlify
зайти в Deploys → Drag & drop a folder вместо предыдущего деплоя.

## Таблицы Supabase

| Таблица | Назначение |
|---|---|
| `formula_entries` | Кормление смесью + вода + стул + фото |
| `medication_entries` | Лекарства (Мальтофер, Фолиевая) |
| `daily_summaries` | Итог дня от няни (самочувствие, сон, комментарий) |
| `weight_entries` | Замеры веса (для графика в профиле) |
| `feeding_entries` | Прикорм (существующая таблица, не трогали) |

Все таблицы открыты публично (`public_all` policy), ничего не
шифруется — это семейный частный URL, не публичный сервис.
